# Roadmap de Implementación

## Fase 0: Fundaciones (1-2 semanas)

Objetivo: preparar infraestructura y base técnica.

Entregables:

- Dominio y subdominios definidos (`www`, `cms`, `staging`).
- Hosting de WordPress provisionado.
- Repositorio y estándar de ramas definido.
- Entornos `dev/staging/prod`.
- CI básico (lint + build).

Criterio de salida:

- WordPress accesible y seguro.
- Astro build desplegando en preview.

## Fase 1: MVP corporativo + Blog (3-5 semanas)

Objetivo: lanzar sitio B2B orientado a captación.

Entregables:

- Homepage de conversión.
- Páginas de servicios e industrias.
- Página de contacto con formulario y tracking.
- Blog listing + blog detail desde WordPress headless.
- SEO técnico base (sitemap, robots, metadata, schema).

Criterio de salida:

- Flujo completo de publicación en WordPress reflejado en web.
- Core Web Vitals dentro de objetivos acordados.

## Fase 2: Optimización comercial y SEO (4-8 semanas)

Objetivo: aumentar leads y autoridad orgánica.

Entregables:

- Plantillas para landings por industria.
- Biblioteca de casos de éxito.
- Calendario editorial y cluster de contenidos.
- Integración con CRM para lead routing.
- Dashboards de conversión.

Criterio de salida:

- Sistema estable de generación y medición de leads.

## Fase 3: Preparación ecommerce (3-6 semanas)

Objetivo: habilitar catálogo sin activar checkout completo.

Entregables:

- WooCommerce instalado en `cms`.
- Modelo de producto y categorías definido.
- PDP (product detail page) en Astro consumiendo APIs.
- Reglas de inventario/precio (si aplica).

Criterio de salida:

- Catálogo navegable desde frontend.

## Fase 4: Marketplace (6-12 semanas)

Objetivo: habilitar compra online.

Entregables:

- Carrito y checkout.
- Pasarela de pagos.
- Cálculo de envío/impuestos.
- Emails transaccionales.
- Panel operativo básico (pedidos, estados, soporte).

Criterio de salida:

- Primer pedido end-to-end en producción.

## Fase 5: Escala y automatización (continuo)

Objetivo: escalar ventas y operación.

Entregables:

- Automatizaciones de marketing.
- A/B testing de páginas clave.
- Reportes de margen por línea de producto.
- Mejoras continuas de UX y performance.
